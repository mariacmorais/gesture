Cholecystectomy Incision Annotation

This tool shows the final frame automatically on page load so you can annotate right away. You can still replay the clip at any time.

1. Select Clip

Add your own clips in clip-config.js or pass a ?video=<URL> query parameter when embedding this page.

2. Final Frame

Final frame ready.

Draw the incision by touching or clicking once to start and once to end the line.

Select a clip to prepare the final frame…

3. Annotate

Tap or click to place the start of the incision, drag, and release to finish.

4. Submit Annotation

Draw the incision on the frozen frame to enable submission.
